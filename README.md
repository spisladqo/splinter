# splinter

Линтер курсовых работ для пользователей Overleaf, в виде аддона для Firefox.

## Что исправляет линтер:
1. Минусы заменяет на тире (с неразрывным пробелом перед и с обычным после).
2. Минус между двумя числами превращает в два минуса, убирает пробелы.
3. Минусы в математических блоках ($$) не меняет.
4. "Обычные" кавычки заменяет на <<ёлочки>>.
5. Добавляет неразрывный пробел перед ссылкой \cite.
6. Убирает пробел перед сноской \footnote.
7. Исправляет пунктуацию и первую букву в элементах списков.

## Установка

1. Клонируем проект.
2. Переходим в браузере на страницу about:debugging, во вкладку This Firefox.
3. Нажимаем Load Temporary Add-on, выбираем manifest.json проекта.

Теперь аддон доступен с остальными аддонами во вкладке Extensions.
